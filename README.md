## FEE AI - Made by Luis Palma

This repository contains the final project for Tech3500.

## Overview

Fee AI is an intelligent PDF chat assistant that allows users to upload PDF documents and have conversations with an AI character named Fee. Fee will read over uploaded PDFs and answer questions based on her unique personality, background, and perspective.

## Features

- PDF Upload & Processing: Drag or drop and browse to upload PDF files
- Session Management: Create, switch between, and delete chat sessions
- RAG (Retrieval-Augmented Generation): AI answers questions using the content from uploaded PDFs
- Responsive Design: Works on both mobile and desktop devices
- Conversation History: All conversations are saved and can be accessed later

### Tables

- pdfs: Stores PDF metadata (filename, file size, upload timestamp)
- chat_sessions: Manages chat sessions (also includes PDF if it was uploaded during a chat session)
- conversation_history: Stores all user and assistant messages
- pdf_chunks: Stores processed PDF text in chunks for quick retrieval
